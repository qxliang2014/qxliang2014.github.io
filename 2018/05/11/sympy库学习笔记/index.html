python符号运算库SymPy学习笔记

发表于 2018-05-11 更新于 2021-03-18 阅读次数： 6.8k 6 分钟

¶引言

所谓用进废退大致也适用于任何一门计算机语言，比如从一个个小目标、小项目做起， 再比如在学习数学的时候用python作为计算分析和作图的有力工具，用好这个工具一定能帮助我们无论在纯数学还是在工程或者编程本身这些方面取得进步。

¶选择使用SymPy

用编程处理问题，特别是科学计算和数据处理等需要一定的数学基础，而且随着项目推进愈来愈需要更深厚扎实的数学功底，python语言有一整个开源强大的科学计算库，或者称科学计算生态系统，包括Numpy,Scipy,Ipython,matplotlib,pandas以及sympy等等，其中只有sympy是可以使用代数及其他各种数学符号来作运算的（其他库偏向于各种数据值运算和工程计算），据官方说法，它的开发目标就是成为一个全功能的计算机代数系统，这就比较合纯数学爱好者的口味了。

¶安装和导入SymPy

最好的情形是已安装使用了python的开源发行版anaconda，它已经包含了上面提及的所有科学计算库，只需一步升级到最新版本即可，这里使用的版本是1.1.1：

conda update sympy

也可以用python自带的包管理器pip来安装，但是要注意自身python的版本和环境配置，python多版本共存可能造成很大麻烦，建议不要用sudo命令：

python -m pip install --user sympy

当然还可以用Git或各种系统软件包安装命令来安装，请参考官方文档

这里假设已在集成了jupyter notebook的anaconda环境下，安装完后，打开jupyter notebook，输入：

In [1] :from sympy import *

运行后无输出就表示导入成功了。

¶提示

  • 新版的SymPy不包含mpmath模块，mpmath现在是sympy外部的一个python库，如果想用就pip安装一下。
